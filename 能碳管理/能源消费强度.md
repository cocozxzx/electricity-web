# 能耗统计与折标计算功能设计

## 1. 功能概述

依据《综合能耗计算通则》（GB/T 2589）的折算方法，将各层级电力消费量换算为标准煤，按时间周期出具电力折标能耗统计结果。

**口径说明**：本功能计算结果仅含电力消费折标量，不含煤、气、油、热力等其他能源品类，不等同于 GB/T 2589 定义的综合能耗。所有报告和展示界面须显著标注此口径，统计指标统一使用"电力折标能耗"，不使用"综合能耗"。

本功能是能耗查询的衍生计算模块，层级结构、权限模型与能耗查询保持一致，不单独设计权限逻辑。


## 2. 折标系数

### 2.1 系数定义

GB/T 2589 中电力折标系数有两种，系统均支持：

| 系数类型       | 折算值                     | 适用场景               |
| -------------- | -------------------------- | ---------------------- |
| 当量值（默认） | 1 kWh = 0.1229 kgce        | 节能考核、内部能效管理 |
| 等价值         | 以国家统计局当年公布值为准 | 能源统计上报           |

系统默认使用当量值。当量值为固定常数，不随年份变化，无需定期更新。等价值由平台管理员按国家统计局每年发布的最新值统一维护，租户不可自行修改。

### 2.2 系数版本管理

- 系数更新后，历史计算结果锁定更新前的系数，不随新系数重算
- 每条计算结果均记录所用系数类型、系数值和 GB/T 2589 版本号，报告中一并展示


## 3. 计算逻辑

### 3.1 基本公式

```
电力折标能耗（kgce）= 用电量（kWh）× 折标系数（kgce/kWh）
电力折标能耗（tce） = 电力折标能耗（kgce）÷ 1000
```

### 3.2 层级聚合

与能耗查询一致，下级之和等于上级：

- 设备折标量之和 = 项目折标量
- 项目折标量之和 = 代理商折标量
- 代理商折标量之和 = 全网折标量

### 3.3 统计周期

支持日、月、季度、年四个周期，也支持自定义起止日期。


## 4. 功能内容

### 4.1 折标计算与展示

- 在能耗查询页基础上，新增"折标视图"切换入口
- 切换后，所有用电量数值同步换算为 tce 展示，图表纵轴单位随之变化
- 保留与能耗查询相同的层级树、时间筛选、趋势图、占比图和明细表

### 4.2 汇总统计表

按时间周期输出结构化统计表，包含以下字段：

| 字段                | 说明                         |
| ------------------- | ---------------------------- |
| 统计对象            | 设备 / 项目 / 代理商 / 全网  |
| 统计周期            | 起止日期                     |
| 用电量（kWh）       | 原始电量数据                 |
| 电力折标能耗（tce） | 折算后标准煤量               |
| 折标系数类型        | 当量值 / 等价值              |
| 折标系数值          | 实际使用的系数               |
| 数据完整率          | 统计周期内原始数据的完整程度 |

### 4.3 折标系数配置

平台管理员可在系统配置页维护折标系数：

- 查看当前系数类型和系数值
- 更新等价值（需填写来源：国家统计局发布年份）
- 查看历史系数版本记录

租户用户无系数配置权限，使用平台统一维护的系数值。

### 4.4 报告生成

支持导出电力折标能耗统计报告，格式为 PDF 或 Excel。

报告必须包含以下内容：

- 口径声明：显著标注"本报告统计口径仅含电力消费折标量，不含其他能源品类，不等同于 GB/T 2589 综合能耗"
- 报告主体公司名称
- 统计周期
- 所用折标系数类型、系数值及 GB/T 2589 版本号
- 各层级电力折标能耗汇总数据
- 数据完整率说明


## 5. 与能耗查询的关系

本功能复用能耗查询的数据基础和页面框架，差异点如下：

| 对比项   | 能耗查询      | 能耗统计与折标计算         |
| -------- | ------------- | -------------------------- |
| 核心指标 | 用电量（kWh） | 电力折标能耗（tce）        |
| 展示单位 | kWh、kW       | tce、kgce                  |
| 报告类型 | 用电明细报告  | 折标能耗统计报告           |
| 系数管理 | 无            | 平台管理员维护折标系数版本 |
| 合规说明 | 无特殊要求    | 必须标注口径声明           |

建议在页面实现上作为能耗查询的衍生视图，通过"折标视图"切换按钮进入，共用左侧层级树和时间筛选栏，不单独开发独立页面。
