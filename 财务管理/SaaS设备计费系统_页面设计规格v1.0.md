# 第一部分 厂家总控台

厂家总控台采用左侧固定导航 + 右侧内容区布局。导航菜单项：数据总览、账单与核销、设备状态查询、参数配置。顶部全局栏显示当前登录账号与退出入口。

 

## P1 数据总览（首页）

登录后默认落地页。提供全局运营视角，快速感知系统整体健康状态，并提供直达高优先级任务的入口。

 

**1.** **页面布局**

  **[** **顶部警告横幅（条件显示） ]**    当「有设备已锁定」或「有欠款超宽限期」时显示红色横幅    文案示例：「当前有 3 台设备处于锁定状态，涉及 2 个项目，点击查看」    点击跳转至 P2 账单与核销，并自动筛选显示相关项目  

 

| **指标卡区（4 列等宽）**                                     |
| ------------------------------------------------------------ |
| 激活设备总数 / 本月新增激活 / 当前锁定设备数 / 当前宽限期设备数 |

 

| **[** **即将到期设备预警列表 ]**    按到期日升序排列    展示字段：设备 ID、所属项目、到期日、剩余天数、对应欠款金额    颜色区分：≤7天红色、8-30天橙色    每行末尾：「查看项目」跳转至对应项目核销 | **[** **欠款代理商排行 ]**    按欠款金额降序    字段：代理商名称、项目数、欠款总额、最久未付天数    点击行跳转至 P2，筛选该代理商 |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
|                                                              |                                                              |

 

**2.** **字段与数据说明**

| **指标卡**       | **数据来源**                    | **说明**               |
| ---------------- | ------------------------------- | ---------------------- |
| 激活设备总数     | device 表 state != 未激活 count | 所有已激活设备，含锁定 |
| 本月新增激活     | activation_time 本月区间 count  | 自然月统计             |
| 当前锁定设备数   | device 表 state = LOCKED count  | 实时值，红色高亮显示   |
| 当前宽限期设备数 | device 表 state = GRACE count   | 实时值，橙色高亮显示   |

 

**3.** **交互说明**

| **交互元素**                       | **触发动作**         | **目标页面 / 结果** |
| ---------------------------------- | -------------------- | ------------------- |
| 顶部警告横幅（点击）               | 跳转并筛选锁定项目   | P2 账单与核销       |
| 即将到期列表「查看项目」           | 跳转至对应项目核销页 | P2，定位至该项目    |
| 欠款排行表（点击行）               | 跳转并筛选该代理商   | P2 账单与核销       |
| 指标卡「当前锁定设备数」（点击）   | 跳转并筛选 LOCKED    | P4 设备状态查询     |
| 指标卡「当前宽限期设备数」（点击） | 跳转并筛选 GRACE     | P4 设备状态查询     |

 



 

## P2 账单与核销

厂家最高频操作页。提供待核销项目列表、执行核销操作、查看核销历史三个核心功能，通过 Tab 切换组织。

 

**1.** **页面布局**

  **[** **页面顶部筛选栏 ]**    搜索框（按代理商名称 / 项目名称）    下拉筛选：全部 / 有锁定设备 / 仅宽限期 / 已核销    日期范围选择器（针对「核销历史」Tab）  

 

| **[ Tab 1****：待核销（默认激活） ]**    列表项见下方字段表    有锁定设备的项目行用红色左边框高亮 | **[ Tab 2****：核销历史 ]**    按核销时间倒序    可按时间范围筛选    支持导出 Excel |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
|                                                              |                                                              |

 

  **[** **核销操作抽屉（点击「核销」后从右侧滑出） ]**    项目名称 + 代理商名称（只读）    欠款明细列表：每条账单的设备  ID、周期、金额（只读）    欠款总额（加粗显示，只读）    实收金额输入框（必填，系统预填欠款总额，可手动核对）    到账时间选择器（默认今天）    「确认核销」按钮 + 「取消」按钮    确认后展示核销结果：解锁设备列表 + 各设备新到期日  

 

**2.** **待核销列表字段**

| **字段**     | **说明**                   | **备注**          |
| ------------ | -------------------------- | ----------------- |
| 项目名称     | 项目全称                   | 点击展开欠款明细  |
| 代理商       | 所属代理商名称             |                   |
| 欠款金额     | 项目 outstanding_balance   | 红色加粗          |
| 锁定设备数   | state=LOCKED 的设备数量    | >0 时显示红色图标 |
| 宽限期设备数 | state=GRACE 的设备数量     | 橙色显示          |
| 最早到期账单 | 欠款中最早那条账单的到期日 | 反映拖欠时长      |
| 操作         | 「核销」按钮               | 欠款=0 时置灰     |

 

**3.** **核销历史列表字段**

| **字段**       | **说明**                                     |
| -------------- | -------------------------------------------- |
| 核销时间       | 服务端核销时间戳                             |
| 项目名称       | 所属项目                                     |
| 代理商         | 所属代理商                                   |
| 核销金额       | 本次核销总金额                               |
| 解锁设备数     | 本次触发解锁的 LOCKED 设备数                 |
| 结清宽限设备数 | 本次结清的 GRACE 设备数                      |
| 操作人         | 厂家操作账号                                 |
| 操作           | 「查看详情」展开解锁设备列表与各设备新到期日 |

 

**4.** **核销操作交互流程**

| **步骤**                         | **系统行为**                                             |
| -------------------------------- | -------------------------------------------------------- |
| ① 点击「核销」                   | 右侧抽屉滑出，展示欠款明细，预填实收金额                 |
| ② 厂家核对金额，点击「确认核销」 | 系统校验实收金额 = outstanding_balance，不符则提示错误   |
| ③ 校验通过                       | 系统记录核销时间戳，欠款余额清零，账单标记「已核销」     |
| ④ 触发解锁                       | LOCKED 设备收到解锁指令，新到期日 = 核销当天 + N 周期    |
| ⑤ 结清宽限期设备                 | GRACE 设备账单结清，宽限期终止，到期日顺延               |
| ⑥ 展示结果                       | 抽屉内展示：解锁设备列表 + 各设备新到期日 + 操作成功提示 |
| ⑦ 刷新列表                       | 该项目从「待核销」Tab 移至「核销历史」Tab                |

 

**5.** **状态与边界说明**

| **核销边界规则**                                             |
| ------------------------------------------------------------ |
| · 实收金额必须精确等于欠款总额，系统强校验，不允许超额或不足 |
| · 欠款为 0 的项目不显示「核销」按钮                          |
| · 核销为不可撤销操作，确认弹窗需二次确认                     |
| · 若核销时设备离线，解锁指令挂起，设备重新联网时自动拉取执行 |
|                                                              |

 



 

## P3 参数配置

低频页面，仅超级管理员可操作。集中管理系统全局计费参数，所有配置变更均记录操作人和时间。

 

**1.** **页面布局**

  **[** **页面标题区 ]**    「参数配置」标题 + 最后修改人 + 最后修改时间（只读提示）  

 

  **[** **分区一：设备定价配置 ]**    表格形式展示各设备类型定价    字段：设备类型 / 日费率（元）/ 年费标准（元）/ 生效时间 / 操作（编辑）    底部「新增设备类型」按钮    编辑时弹出行内编辑或模态框，填写新定价后点击「保存」    保存后提示：「定价变更仅对新激活设备生效，存量设备不受影响」  

 

  **[** **分区二：宽限期配置 ]**    单行输入：宽限天数（数字输入框，1-30 天）    当前值醒目显示，修改后点击「保存」    保存后提示：「变更对下一个到期周期生效」  

 

  **[** **分区三：激活异常阈值 ]**    单行输入：距出货日超 N 天未激活则告警（数字输入框）    说明文案：「超过阈值的设备将在数据总览页产生告警，需人工核定」  

 

  **[** **底部：配置变更历史 ]**    折叠展示，默认收起    字段：变更时间 / 变更项 / 变更前值 / 变更后值 / 操作人  

 

**2.** **字段说明**

| **配置项**   | **类型**       | **校验规则** | **说明**                                   |
| ------------ | -------------- | ------------ | ------------------------------------------ |
| 设备日费率   | 数字，精确到分 | > 0          | 变更仅影响新激活设备                       |
| 设备年费标准 | 数字，精确到元 | > 0          | 与日费率二选一或并存，厂家可自定义展示哪种 |
| 宽限期天数   | 整数           | 1 ≤ N ≤ 30   | 全局统一，所有设备适用                     |
| 激活异常阈值 | 整数（天）     | 1 ≤ N ≤ 365  | 触发告警，不自动锁定                       |

 

**3.** **交互说明**

| **交互元素**              | **触发动作**                                                 |
| ------------------------- | ------------------------------------------------------------ |
| 「编辑」按钮（定价行）    | 该行变为可编辑状态，显示「保存」「取消」                     |
| 「保存」（定价）          | 弹出确认框：「此操作仅影响新激活设备，是否确认？」→ 确认后保存 |
| 「保存」（宽限期 / 阈值） | 直接保存并刷新当前值显示                                     |
| 「新增设备类型」          | 弹出模态框填写类型名称和定价                                 |
| 「配置变更历史」折叠箭头  | 展开 / 收起历史表格                                          |

 



 

## P4 设备状态查询

与已有「设备管理」互补。设备管理负责设备基础信息 CRUD；本页以「状态」为第一维度，供厂家快速定位异常设备并进行应急操作。

 

**1.** **页面布局**

| **[** **顶部状态 Tab ]**    全部 / LOCKED /  GRACE / ACTIVE    Tab 上显示各状态设备数量角标 | **[** **筛选栏 ]**    代理商下拉 / 项目下拉（联动）    到期日范围选择器    设备 ID 搜索框 | **[** **批量操作栏（选中设备后显示）  ]**    「批量锁定」「批量解锁」按钮    仅应急使用，操作需二次确认 |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
|                                                              |                                                              |                                                              |

 

  **[** **设备列表（主体区域） ]**    字段见下方字段表    支持按到期日、状态变更时间排序    每行末尾：「锁定」/「解锁」单设备操作按钮  

 

**2.** **列表字段**

| **字段**   | **说明**                | **备注**               |
| ---------- | ----------------------- | ---------------------- |
| 设备 ID    | 设备唯一标识            |                        |
| 设备类型   | 网关 / 子设备           |                        |
| 所属项目   | 项目名称                | 点击跳转至 P2 对应项目 |
| 所属代理商 | 代理商名称              |                        |
| 激活时间   | 首次联网时间            |                        |
| 到期日     | 当前到期日              | 过期日期标红           |
| 剩余天数   | 到期日 - 今天           | 负数=已超期，标红      |
| 当前状态   | ACTIVE / GRACE / LOCKED | 色标区分               |
| 状态版本号 | state_version           | 调试用，默认折叠       |
| 操作       | 锁定 / 解锁             | 按当前状态展示可用操作 |

 

**3.** **手动锁定 / 解锁交互**

| **应急操作说明**                                             |
| ------------------------------------------------------------ |
| · 手动锁定：仅在系统自动锁定未能及时触达时使用（如设备长期离线），需填写操作原因 |
| · 手动解锁：仅在核销流程之外的特殊情况使用（如厂家内部测试），需填写操作原因并由管理员审批 |
| · 所有手动操作均记录操作人、时间、原因，不可撤销             |
| · 手动解锁不自动更新到期日，到期日更新仅由核销流程触发       |
|                                                              |

 



 

# 第二部分 代理商门户

代理商门户同样采用左侧导航 + 右侧内容区布局。导航菜单：项目总览、到期预警、定价查看。项目详情、设备列表通过项目总览页跳转进入，不单独列于导航。

 

| **权限说明**                                                 |
| ------------------------------------------------------------ |
| · 代理商只能看到自己名下的项目和设备，不可看到其他代理商数据 |
| · 代理商无任何设备操作权限（锁定 / 解锁 / 激活时间修改均不可操作） |
| · 定价页面只读，不可修改任何参数                             |
|                                                              |

 

## P5 项目总览（首页）

登录后默认落地页。代理商一眼看清所有项目的健康状态，快速定位需要处理的项目。

 

**1.** **页面布局**

  **[** **顶部警告横幅（条件显示） ]**    有锁定设备时：红色横幅，文案「有 N 台设备已锁定，请尽快付款」，点击跳转至金额最大的欠款项目    有设备进入宽限期时：橙色横幅，文案「有 N 台设备进入宽限期，剩余 X 天」  

 

| **[** **汇总指标卡 ]**    我的项目数    锁定设备总数（红色）    宽限期设备总数（橙色） | **[** **项目列表 ]**    （见下方字段表） | **[** **快捷操作栏 ]**    「查看到期预警」按钮    跳转至 P8 |
| ------------------------------------------------------------ | ---------------------------------------- | ----------------------------------------------------------- |
|                                                              |                                          |                                                             |

 

**2.** **项目列表字段**

| **字段**   | **说明**             | **交互**                                |
| ---------- | -------------------- | --------------------------------------- |
| 项目名称   | 项目全称             | 点击进入 P6 项目详情                    |
| 设备总数   | 项目下所有设备数     |                                         |
| 运行中     | state=ACTIVE 数量    | 绿色                                    |
| 宽限期中   | state=GRACE 数量     | 橙色，>0 时高亮                         |
| 已锁定     | state=LOCKED 数量    | 红色，>0 时高亮                         |
| 当前欠款   | outstanding_balance  | 红色加粗，=0 显示「-」                  |
| 最近到期日 | 最近一台设备的到期日 | 提示代理商关注                          |
| 操作       | 「查看详情」         | 跳转 P6；有欠款时改为「立即付款」并高亮 |

 

**3.** **列表状态色标**

| **行状态** | **触发条件**                 | **视觉表现**                                     |
| ---------- | ---------------------------- | ------------------------------------------------ |
| 正常       | 无欠款、无锁定、无宽限期设备 | 默认样式                                         |
| 需关注     | 有设备进入宽限期（无锁定）   | 行左边框橙色                                     |
| 需立即处理 | 有设备已锁定                 | 行左边框红色，「操作」列改为红色「立即付款」按钮 |

 



 

## P6 项目详情 / 账本

代理商门户最核心的页面。展示项目账本全貌，承载付款操作的完整流程。

 

**1.** **页面布局**

  **[** **页面顶部：项目基本信息栏 ]**    项目名称（大字）+ 所属代理商    四个指标卡：设备总数 / 锁定设备数 / 宽限期设备数 / 当前欠款额    当前欠款 > 0 时：右侧显示红色「立即付款」大按钮；欠款 = 0 时按钮置灰  

 

| **[** **左栏：锁定 / 宽限期设备清单（条件显示） ]**    有锁定或宽限期设备时展示此区域    列表字段：设备 ID / 类型 / 到期日 / 状态 / 本条账单金额    底部显示：「解锁所需付款总额：XXX  元」    点击设备 ID 跳转至 P7 设备列表并定位该设备 | **[** **右栏：账本流水时间线 ]**    按时间倒序排列所有账本事件    事件类型 1：账单追加（设备到期出账）    事件类型 2：核销完成（厂家操作）    每条事件展示：时间 / 事件类型 / 涉及设备 / 金额变化    支持按事件类型筛选 |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
|                                                              |                                                              |

 

  **[** **底部：付款操作区（欠款>0时展示） ]**    展示当前应付总额（等于  outstanding_balance）    上传付款凭证按钮（支持图片 /  PDF）    凭证上传后显示「待厂家审核」状态    若凭证已提交未核销：显示「凭证审核中」提示，隐藏重复提交入口  

 

**2.** **账本流水事件类型详解**

| **事件类型** | **显示内容**                                     | **金额变化展示**           |
| ------------ | ------------------------------------------------ | -------------------------- |
| 账单追加     | 设备 ID + 设备类型 + 计费周期（起止日）          | + XXX 元（红色）           |
| 核销完成     | 核销时间 + 核销金额 + 操作人（厂家账号）         | - XXX 元（绿色，欠款减少） |
| 设备解锁     | 解锁设备 ID + 新到期日（核销事件下的子条目）     | —                          |
| 宽限期结清   | 结清设备 ID + 到期日顺延说明（核销事件下子条目） | —                          |

 

**3.** **付款流程交互**

| **步骤**   | **代理商操作**                              | **系统响应**                                     |
| ---------- | ------------------------------------------- | ------------------------------------------------ |
| ① 查看欠款 | 进入项目详情页，查看欠款总额和明细          | 展示 outstanding_balance 及各账单条目            |

## P9 定价查看

只读页面。帮助代理商了解当前定价标准，以便预估未来账单金额。无任何编辑权限。

 

**1.** **页面布局**

  **[** **页面顶部提示栏 ]**    灰色提示：「以下为厂家配置的计费标准，仅供参考，代理商不可修改」

页面底部提供简单的账单预估工具，帮助代理商规划打款节奏。

  **[** **账单预估计算器 ]**    输入：网关数量 + 子设备数量 + 计费周期    输出：预计年费总额    仅供参考，实际以账单为准，计算器不产生任何业务数据
